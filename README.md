# Employee Salary Calculator

A Python application that calculates employee salaries 
including overtime pay.

## Features
- Enter employee name
- Enter hours worked and hourly rate
- Automatically calculates overtime at 1.5x rate 
  for hours exceeding 160
- Calculates normal salary for standard hours
- Displays full salary summary in Rands (R)

## Technologies Used
- Python 3

## How To Run
1. Download the file
2. Open in Python IDLE or any code editor
3. Run the program
4. Enter employee name, hours worked and hourly rate
5. View your salary summary
